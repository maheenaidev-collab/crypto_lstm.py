# AI Crypto Price Predictor (LSTM)

An AI-based cryptocurrency price prediction project built using deep learning.
This project uses an LSTM (Long Short-Term Memory) neural network to analyze historical crypto market data and predict future prices.

## Features

* Deep Learning price prediction
* Bitcoin historical data analysis
* Time-series forecasting
* Data visualization with charts

## Technologies Used

* Python
* TensorFlow
* NumPy
* Pandas
* Matplotlib
* Yahoo Finance API

## Installation

Install required libraries:

pip install tensorflow pandas numpy yfinance matplotlib scikit-learn

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/crypto-lstm-predictor

2. Go to the project folder

cd crypto-lstm-predictor

3. Run the script

python crypto_lstm.py

## Example Output

The model trains on historical Bitcoin price data and predicts future values using an LSTM neural network.
